📘 README — Manheim Recon Asset & PM Tracker
🔧 Manheim Recon Shop
Asset & PM Tracker
v2.0 · Latest Vanilla HTML/JS ● Active No Dependencies localStorage

A single-file, zero-dependency web application built to Manheim brand standards for tracking recon shop equipment and preventive maintenance (PM) schedules across all U.S. auction locations. No build step, no framework, no server required.

⚡
Zero-install deployment. Drop recon_asset_manager.html anywhere — SharePoint, Teams tab, local file system, or any static host. Open in any modern browser and it works instantly. All data is stored locally in the browser.
✨ Features
📊
Live Dashboard
Real-time KPI cards: total assets, active count, out-of-service, PM status breakdown, and active location count with searchable table.
📍
73 Locations
All Manheim U.S. auction sites. Filter by location and shop type — Mechanic, Body & Paint, or Detail.
🔔
PM Alerts
Color-coded alerts: green (current), yellow (due ≤30 days), red (overdue). Visual countdown progress bars per asset.
📝
Full CRUD
Add, edit, and delete equipment records. Inline edit modal for PM updates without leaving the page.

💾 Data Persistence

All asset records are stored in the browser's localStorage under the key manheim_recon_assets_v1. Data persists across page refreshes on the same device and browser profile.

🌱
Seed data included. A sample dataset for Manheim Atlanta is automatically loaded on first launch — covering all three shop types and demonstrating all PM status states (current, due soon, overdue).

🚀 Getting Started
  1. Open recon_asset_manager.html in any modern browser (Chrome, Edge, Firefox, Safari).
  2. Navigate to Locations, select your Manheim site from the dropdown.
  3. Click + Add Asset to log equipment with its last PM date and service interval.
  4. Use the Dashboard to view all assets with PM status across all locations at a glance.
  5. Click Edit on any row to update the PM record after service is completed.
